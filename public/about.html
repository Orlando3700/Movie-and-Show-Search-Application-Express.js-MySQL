About This App

Welcome to the Movies & Shows Explorer! This application allows you to search for your favorite movies and TV shows using real-time data from a public API. You can browse through results, view detailed info, and even save your favorites.

This app was created as a personal project to practice front-end and back-end web development using HTML, CSS, JavaScript, Express.js, API, and a MySQL database. The goal is to provide a simple and clean movie discovery experience.
